Glossary of Terms

There are many terms that can be used in Web Development. Here are 50 words to increase your knowledge and learn new stuff !

Web Development - Web Development mainly talks about tasks associated with developing websites. These can include web design and making applications with JavaScript.

HTML - Hypertext Markup Language. HTML is the language used to create webpages.

HTML Element - A HTML element is an individual component of an HTML document.

HTML5 - HTML5 is the newest and most up to date version of HTML.

ID's - Id's make a unique id for an HTML element.

HREF - Stands for Hypertext Reference. It is the HTML code used to create a link to another page.

CSS - Cascading Style Sheets. They describe how HTML elements should be displayed on the screen.

The Box Model - The Box Model is used when talking about design and layout. The box model is basically a box that wraps around every HTML element.

Classes - The class attribute is mostly used to point to a class in a style sheet.

Anchor Text -Anchor text is the clickable text in a hyperlink.

Hyperlink - Usually an icon or line of text that links to another file or object.

Hypertext - What a Hyperlink is made up of. It is made up of words that contain a link to a website.

Header - The header is located at the top of a website. It usually includes the company logo and the navigation bar. It is kept the same over all webpages

Footer - The footer of a webapge is located at the bottom. It typically contains contact information and authorship information.

Embedded - Content that has imported another resource into the document. For example, you can have embedded text

Flex Box - FlexBox is a layout that was designed to encorpate lots of different screen sizes and was developed as part of the mobile first initiative.

Syntax - The grammar, structure, or order of the elements.

Float - This is a CSS property. It allows an image to 'Float' next to a piece of text.

Display - This is another CSS property. It specifies the type of rendering box used for an element. It specifies how the item will be displayed

Attributes - Attributes are written in HTML and can belong to any element.Theyprovide additional information about an element.

Parent and Child - In CSS, you can have parent and child tags. The paren will tell the child what to do and they do it.

RWB Responsive Web Design -Simmilar to FlexBox, RWB is an approach to web design which makes web pages render well on a variety of devices.

Material Design - This is a design language developed by Google. It focuses on grid-based layouts, responsive animations and transitions.

Markup - A markup language is a computer language that uses tags to define elements within a document.

Containers - This is used in HTML, the container is the area enclosed by the beginning and ending tags.

Nested Elemnets - This means that tags that have been opened should be the next one to close. So it is then in a kind of nest.

Selector - In CSS, selectors are patterns used to select the element you want to style.

Validator - A validator is a program used to check HTML code for syntax errors. Two examples are W3C and WHATWG

W3C - The W3C is a validator founded in 1994 by Tim Berners-Lee, the original architect of the World Wide Web.

WHATWG - WHATWG stands for Web Hypertext Application Technology Working Group. It is a community of people interested in evolving HTML and related technologies.

Decleration - Decleration for example can be a doctype. This will declare to the browser what type of HTML is used. This is only 1 example.

Javascript - An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Contrast - The change in the appearance of a colour surrounded by another colour. In Web Development, colour contrast is very important if you want to make an efficient web page. You wouldnt want to use contrasting colours, as it makes things hard to see.

Feathering - Feathering creats a soft edge around a selection It is a way to blend images, highlight an object within an image. In Web Development and websites it is used for these purposes and to adjust harsh edges for web graphics.

JPEG - The JPEG is the most common image file used on the internet. It is a format for compressing image files.

GIF's - This is a file extension for an animated graphics file. It is the second most common image format used on the World Wide Web after JPEG.

Kerning - This will adjust the spacing between (characters) in a piece of text to be printed.

HEX code - A color hex code is a way of specifying color using hexadecimal values. This can be used instead of RGB code.

Vector Graphics - Vector graphics are created in graphics packages and consist of shapes called objects. Vector graphics are scalable - ie when you resize them, they do not lose quality.

Negative Spacing - This is the space around and between the subject of an image.

PPI - Stands for Pixels Per Inch. It is a measure of the resolution, or clarity, of a video display. The higher the number of pixels per inchthe more detail the screen will convey.

Favicon - The Favicon is an image usually in the top left of a web browser page.

Serif - A serif is a small line attached to the end of a stroke in a letter or symbol. A typeface without serifs is called sans-serif. This is very important in web design as if you pick a wrong font its very hard to read or interpret.

Resolution - This is the number of horizontal and vertical pixels on a display screen.

Watermark - A faint design made in some paper during manufacture that is visible when held against the light and typically identifies the maker.

Contrast Ratio - This is the ratio of the luminance of the brightest color (white) to that of the darkest color (black) A high contrast ratio is better for a screen.

Flat Design - This is another design concept. It focuses on emphasizing minimum use of stylistic elements that give the illusion of three dimensions. It should be simple elements without drop shadows qne other 3D effects.

Radio Button - A radio button is a type of selection button with a GUI interface.

AJAX - AJAX. This is a technique for creating fast and dynamic web pages. It allows web pages to be updated in small parts. This means you can edit some parts of a page without having to reload the whole page.

CMYK - This is another colour scheme and picker. It stands for cyan, magenta, yellow, and key or black.
